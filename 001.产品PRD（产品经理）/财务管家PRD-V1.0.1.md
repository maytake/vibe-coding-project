## 1. 产品概述

### 1.1 产品背景

在日常生活中，个人和家庭往往缺乏系统性的理财记录工具。大多数人通过微信支付、支付宝、银行APP查看消费记录，但这些工具分散且不直观，难以对消费行为进行有效分析和规划。小微企业和个体户更需要清晰的收支记录来了解经营状况。

### 1.2 产品定位

**财务管家** 是一款轻量级的个人/家庭财务管理工具，帮助用户快速记录日常消费/收入，提供直观的统计分析和财务报表，让用户清晰地了解钱花在哪里、赚了多少。

### 1.3 核心价值主张

- **简单高效**：3秒内完成一笔记账，告别繁琐流程
- **清晰可见**：消费分类统计，每月收支一目了然
- **安全可靠**：用户注册登录，数据独立管理
- **多人协作**：支持家庭/情侣共享账本，共同管理财务

### 3.2 详细功能描述

#### 3.2.1 用户注册

- **入口**：登录页"注册账号"链接
- **流程**：
  1. 输入用户名（3-20字符，仅中文/字母/数字/下划线）
  2. 输入密码（6-20字符）
  3. 再次输入密码确认
  4. 点击注册
- **校验**：
  - 用户名不能为空
  - 用户名3-20字符
  - 密码6-20字符
  - 两次密码一致
  - 用户名不能已存在
- **反馈**：注册成功自动登录，跳转首页

#### 3.2.2 用户登录

- **入口**：首页未登录时显示登录页
- **流程**：
  1. 输入用户名
  2. 输入密码
  3. 点击登录
- **校验**：
  - 用户名和密码不能为空
  - 用户名密码匹配
- **反馈**：登录成功跳转首页，欢迎用户

#### 3.2.3 快速记账

- **入口**：首页大按钮「+」，点击直接进入记账页面
- **流程**：
  1. 选择收支类型（支出/收入）
  2. 选择分类（图标+名称）
  3. 输入金额（数字键盘）
  4. 可选：选择账户、添加备注、选择日期
  5. 点击保存
- **反馈**：保存成功 toast 提示，返回首页列表自动更新

#### 3.2.4 分类体系

**支出默认分类**（10个）：
- 🍚 餐饮
- 🚗 交通
- 🛒 购物
- 🎬 娱乐
- 🏠 住房
- 💊 医疗
- 📱 通讯
- 👔 服饰
- 🎁 人情
- 📦 其他

**收入默认分类**（5个）：
- 💰 工资
- 💵 兼职
- 🎯 奖金
- 🎁 礼金
- 📥 其他

#### 3.2.5 账户管理

支持的账户类型：
- 💵 现金
- 💳 银行卡
- 📱 微信支付
- 🟢 支付宝
- 📘 银行账户

每个账户可设置：
- 账户名称（可自定义）
- 账户类型
- 初始余额
- 颜色/图标

#### 3.2.6 统计报表

**月度概览**：
- 本月收入总计
- 本月支出总计
- 本月结余（收入-支出）
- 与上月对比（上升/下降百分比）

**收支明细**：
- 按日期排列的记账列表
- 支持筛选（按分类、按账户、按日期范围）

**分类统计**：
- 饼图展示各分类占比
- 点击分类可查看该分类下的详细记录

#### 3.2.7 数据导出

- 导出格式：Excel（.xlsx）
- 导出内容：日期、类型、分类、账户、金额、备注
- 筛选能力：可选择日期范围

## 4. 非功能需求

### 4.1 性能要求

- 页面首次加载 ≤ 2秒
- 记账保存响应 ≤ 500ms
- 支持10000条记账数据，查询不卡顿

### 4.2 兼容性

- **浏览器**：Chrome、Safari、Edge 最新版
- **手机浏览器**：iOS Safari、Android Chrome 及微信内置浏览器
- **屏幕适配**：响应式布局，支持手机竖屏（375px-428px）和平板横屏

### 4.3 安全要求

- 用户密码本地存储（加密存储）
- 用户数据按用户分离（localStorage按用户隔离）
- 登出后清除本地会话标识
- 敏感数据不记录身份证号、银行卡号等

### 4.4 辅助功能

- 深色模式支持（跟随系统设置）
- 无障碍支持：按钮有足够的点击区域（至少44x44px）
